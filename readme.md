# Laravel 5.2 Boilerplate

## Features

- Laravel 5.2
- Barryvdh
  - IDE Helper
  - Debugbar
  - DomPDF
- Whoops
- Bootstrap 3.3.6
- Font Awesome 4.6.3

## Installation

Clone Project

```sh
git clone <repository> <foldername>
cd <foldername>
composer install
[sudo] npm i -g gulp
npm install
npm run dev
```

Open your browser and point to **http://homestead.app:3000** to enable **Browsersync**
